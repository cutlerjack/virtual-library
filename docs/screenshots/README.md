# Screenshot Baselines

Store visual regression screenshots here. Recommended captures after each design phase:

- `home-light.png` / `home-dark.png` — Homepage with shelf
- `post-light.png` / `post-dark.png` — A post page with drop-cap, sidenotes
- `archive-light.png` / `archive-dark.png` — Archive catalogue view
- `tags-light.png` / `tags-dark.png` — Tag garden
- `404-light.png` / `404-dark.png` — Error page
- `mobile-home.png` — Homepage at 375px width

Naming convention: `{page}-{theme}-{date}.png` (e.g., `home-light-2026-03-06.png`)
